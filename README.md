## 프로젝트 개요 (2020년도 2학기)
- 명칭
  SmartCaller = Smart + Caller
- 개요
  거울을 보며 동시에 일정 확인과 날씨 정보를 확인하여 바쁜 일상을 사는 현대인들의 아침을
  효과적으로 시작할 수 있도록 도와줍니다.
